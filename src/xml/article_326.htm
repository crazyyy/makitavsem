  • Мощный асинхронный двигатель

  • Удобный держатель для аксессуаров

  • Катушка для шланга высокого давления
  • Автоматический предохранительный перепускной клапан
  • Двойной бак для моющих средств (0,7 л x 2)
  • Автоматическая система TSS  включает/выключает электродвигатель путем нажатия пистолетной рукоятки
  • Прочный корпус гидронасоса из латуни
  • Для удобной транспортировки мини-мойки установлены рукоятка с резиновыми вставками и большие колеса с  протектором в основании корпуса
  • Регулировка давления воды

  • Встроенная катушка
  • Удлинительная трубка с насадкой
  • Шланг высокого давления
