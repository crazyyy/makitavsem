Пилы по металлу Makita

Отрезная пила по металлу Makita LW1401 (2200Вт; 355мм; 3800об/мин; 16,6кг)

Монтажная пила Makita LW1401 отличается высоким уровнем эргономики благодаря продуманной конструкции рукоятки. Есть защита оператора от искр при обработке металла. Возможность наклонного реза, изменения его глубины и простая регулировка за счет быстрозажимного приспособления. D-образная рукоятка для удобства захвата.
Мощный двигатель для резки металла, цветных металлов, ПВХ.
Защитный кожух для защиты оператора от искр.
Регулируемая направляющая пластина (45° вправо, 3 положения).
Регулируемый ограничитель глубины реза.
Быстрозажимное приспособление для установки угла резания.
Быстрая смена диска.

Комплектация:

  • Торцовый ключ SW17
  • Твердосплавный диск
