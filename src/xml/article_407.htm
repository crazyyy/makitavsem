Пылесос Makita VC2012L (1000Вт; 8.5кг)

  • Пылесос Makita VC2012L предназначен для сухой и влажной уборки
  • Контейнер для мусора выполнен из нержавеющей стали, что гарантирует долгий срок службы
  • Данное устройство может работать в паре с электроинструментом мощностью до 2600 Вт
  • При подключении шланга к специальному отверстию для выдува воздуха, может использоваться как воздуходувка
  • В комплект входит всё необходимое для комфортной работы

Комплектация:

  • шланг 3.5 м
  • резиновый шнур питания 7.5 м
  • фильтроэлемент
  • оконечник шланга съемный с регулятором силы всасывания
  • труба — 2 шт.
  • универсальная щетка для сухой уборки
  • щелевая насадка
  • насадка с щеткой и резиной
  • текстильный мешок
  • пластиковый мешок для мусора
  • крепеж для трубы на корпус
  • адаптер для подключения к инструменту с регулировкой силы всасывания
  • ящик для инструмента
